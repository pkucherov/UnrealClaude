```cpp
/** Production HTTP client using UE's FHttpModule (D-20, D-22) */
class FOpenCodeHttpClient : public IHttpClient
{
public:
    FOpenCodeHttpClient() = default;
    virtual ~FOpenCodeHttpClient() = default;

    // IHttpClient interface
    virtual int32 Get(const FString& Url, FString& OutBody) override;
    virtual int32 Post(const FString& Url, const FString& Body, FString& OutResponse) override;
    virtual void BeginSSEStream(const FString& Url,
        TFunction<bool(void* Ptr, int64 Length)> StreamCallback,
        TFunction<void()> OnDisconnected) override;
    virtual void AbortSSEStream() override;

    /** Convenience: check OpenCode server health. Returns true if healthy. */
    bool CheckHealth(const FString& BaseUrl, FString& OutVersion);

private:
    /** Active SSE request (for cancellation) */
    TSharedPtr<IHttpRequest, ESPMode::ThreadSafe> ActiveSSERequest;
};
```

**Create OpenCodeHttpClient.cpp:**

Implement all methods:

1. **Get()**: Create IHttpRequest via `FHttpModule::Get().CreateRequest()`. Set URL, verb GET. Use `FEvent` to synchronize: create an `FEvent*` with `FPlatformProcess::GetSynchEventFromPool()`, bind `OnProcessRequestComplete` to set result and trigger the event, call `ProcessRequest()`, wait on the event with a 30-second timeout, return status code and body from the response. Log URL and status code at Verbose level.

2. **Post()**: Same pattern as Get, but set verb POST, set `Content-Type: application/json` header, set content body.

3. **BeginSSEStream()**: Create request, set URL, verb GET, headers `Accept: text/event-stream` and `Cache-Control: no-cache`. Call `SetTimeout(0.0f)` (CRITICAL per research — prevents UE from closing long-lived SSE). Set stream delegate via `SetResponseBodyReceiveStreamDelegate` that forwards chunks to `StreamCallback`. Bind `OnProcessRequestComplete` to call `OnDisconnected` via `AsyncTask(ENamedThreads::GameThread, ...)`. Store request in `ActiveSSERequest`. Call `ProcessRequest()`.

4. **AbortSSEStream()**: If `ActiveSSERequest` is valid, call `CancelRequest()` and reset the pointer.

5. **CheckHealth()**: Call `Get(BaseUrl / TEXT("global/health"), Body)`. If status != 200, return false. Parse JSON body, extract `healthy` bool and `version` string. Return healthy.

Log format: `UE_LOG(LogUnrealClaude, Verbose, TEXT("OpenCodeHTTP %s %s → %d"), *Verb, *Url, StatusCode)`.

IMPORTANT: The sync Get/Post pattern using FEvent must handle the case where the HTTP module is not available (editor shutting down). Check `FHttpModule::IsAvailable()` first. If not available, return -1.
